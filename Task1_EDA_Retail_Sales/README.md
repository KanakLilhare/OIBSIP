
# Exploratory Data Analysis on Retail Sales Data

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the Supermart Grocery Sales dataset to uncover sales trends, customer behavior, product performance, and business insights.

## Dataset
Supermart Grocery Sales - Retail Analytics Dataset

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Analysis Performed
- Data Cleaning
- Descriptive Statistics
- Time Series Analysis
- Customer Analysis
- Product Analysis
- Regional Analysis
- Correlation Analysis
- Data Visualization

## Key Insights
- Eggs, Meat & Fish generated the highest sales.
- Oil & Masala recorded the lowest sales.
- Top customers contributed significantly to revenue.
- Sales showed monthly fluctuations.
- Regional sales performance varied considerably.

## Recommendations
- Focus inventory on high-performing categories.
- Implement customer loyalty programs.
- Optimize discount strategies.
- Increase promotional efforts for lower-performing categories.
